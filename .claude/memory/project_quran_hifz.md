Building a Quran memorization web app with guided learning phases.

**Why:** Existing apps skip the learning process — they're verification tools, not teaching tools. This app uses research-backed techniques: listen-first, chunking, semantic understanding, and spaced repetition.

**How to apply:**
- Tech: Next.js 14+ (App Router), Tailwind CSS, Zustand, localStorage/IndexedDB for V1
- MVP scope: 5-8 surahs starting with Al-Fatiha, Al-Ikhlas, Al-Falaq, An-Nas, Al-Kawthar
- Design: calm, focused, warm tones (cream bg, deep teal, muted gold), proper Arabic fonts
- Mobile-first, RTL support, offline-capable
- No backend for V1, no auth, no monetization
- Full spec in `.claude/app-spec-context.md`
